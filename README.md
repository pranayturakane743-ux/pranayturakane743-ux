### How to enable the animated snake

1. Create a GitHub repository with the **exact same name** as your username: `pranayturakane743-ux`
2. Inside that repo, create the file `.github/workflows/snake.yml` with the content below:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: '0 */6 * * *'
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: Platane/snk@master
        with:
          github_user_name: pranayturakane743-ux
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark
            dist/ocean.gif?color_snake=blue&color_dots=#bfd6f6,#8dbdff,#64a1f4,#4b91f1,#4c7cf9
          git_isolate: true
          git_push_branch: output
```

3. Commit and push this workflow file
4. After the first run (every 6 hours), the animated snake `.svg` and `.gif` will appear in the `output` branch of your profile repo
5. The `.gif` version (`dist/ocean.gif`) provides the **smooth animated snake** effect — you can swap the `<img>` src above to that `.gif` URL for a fully animated experience
